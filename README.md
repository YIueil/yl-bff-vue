# y-bff-vue

> 基于 ant design vue 1.x 版本实现的基础前端框架

## 框架能力

- [x] 动态皮肤样式定制支持 [\(参考地址\)](https://segmentfault.com/a/1190000016061608)
- [x] 符合标准token存取机制
- [ ] i18n国际化支持
- [x] 标准的请求体响应体定制
- [ ] 动态路由配置
- [x] Vuex实现的用户角色和权限菜单的状态管理
- [ ] 头部请求条支持
- [ ] 加密支持
- [ ] 提供管理端通用的一套组件
- [ ] 移动端适配
